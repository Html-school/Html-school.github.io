HTML basics

let's quickly explain what html is first.

html is a skeleton of a webpage,which means it looks bad but it makes the page work. It works with languages like javascript and css. Javascript makes the page reactive like refreshing only one part of the page etc. and css makes styles for the page like the bar with the tutorials has a darker background + the hover animation and the commands have the cyan outline around them. This is going to be basics of HTML like adding text and images




making the skeleton for the skeleton

<!DOCTYPE html>

<html>

<head>

  <meta http-equiv="CONTENT-TYPE" content="text/html; charset=UTF-8">

  <title>Hello, World!</title>

</head>

<body>

  <h1>

    Hello, World!

  </h1>

</body>

</html>
